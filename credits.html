Intro

It is impossible to transcribe the grandeur of Eric Cooper's work on a webpage. We started this project not because Eric needed more clients. But we felt a strong desire to make it available into the collective memory of human civilization - the Internet. Internet has this wonderful effect of compressing time and distances with a mere click of a mouse. This effect works very well to exponentially multiply the innate nature of the Art - to be more inspiring - the more easily and widely it can be accessed.

Our hope is that you feel the same inspiration and can merge it with our own line of work.

    Accessing source code

    The entire source code for this website is available on Github. Minor content edits can be done directly using the github edit page interface. Select the appropriate file and click the "edit" button on top. To finish changes to a file, add a message to summarise the changes done and click the "save" button. You should ideally use the Zen mode for a fullscreen editing. You need to ignore any surrounding HTML code whilte altering the text content. Thankfully github keeps history of all the changes, so if you make a mistake, all is not lost.

    If you are a developer please consider using Docpad setup to generate static website, instead of directly editing the code of static web pages. Please keep in mind that you would need to merge back any change made to the content on the static webpage before you overwrite any file.
